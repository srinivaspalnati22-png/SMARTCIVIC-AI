## 🚀 Quick Start

1. **Start Server**: `cd backend && python app.py`
2. **Open Browser**: Navigate to `http://127.0.0.1:5000`
3. **Test Features**:
   - Toggle theme (🌓 button)
   - Submit a complaint
   - View admin dashboard
   - Check heatmap
   - Export data to CSV

## 💡 Pro Tips

1. **Use Quick Categories** on report page for faster submission
2. **Toggle Theme** to match your preference (saves automatically)
3. **Search Admin Dashboard** to find specific complaints quickly
4. **Export to CSV** for data analysis in Excel/Sheets
5. **Upload Images** for AI-powered category detection
6. **Enable Voice** for hands-free reporting
